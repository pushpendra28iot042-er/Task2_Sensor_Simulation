# Automatic Street Light Using LDR Sensor

## Objective

To automatically control an LED based on light intensity using an LDR sensor and Arduino.

## Components Used

* Arduino Uno
* LDR Sensor
* LED
* 220Ω Resistor
* 10kΩ Resistor
* Breadboard
* Jumper Wires

## Working Principle

The LDR senses ambient light and sends analog values to Arduino through pin A0. When the light level decreases, the LED turns ON. When the light level increases, the LED turns OFF.
